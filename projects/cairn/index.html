Overview

Cairn is a modern project and task management service inspired by the way cairns—stacks of stones—guide travelers through complex terrain. Just as a cairn marks the path when the way forward is unclear, Cairn helps you navigate and organize even the most complicated projects, one task at a time.

Key Features

📋

Kanban Task Board

Classic kanban board with drag & drop task management between columns.

🔗

Project Flow Diagram

Visualize and manage task dependencies with a drag-and-connect flow diagram.

🚦

Status Management

Tasks remain blocked until all dependencies are complete. Color coding instantly shows which tasks are ready for work and which are in progress.

☁️

Cloud Storage

All projects and tasks are securely stored in the cloud for access anywhere.

👥

Team Collaboration

Invite team members to your projects to work on large projects together.

⚡

Parallel Workflows

Clear dependency tracking helps teams identify independent tasks that can be worked on simultaneously.

Technical Details

Vue 3

Modern reactive framework for building dynamic web applications.

Vue Flow

Powerful library for building interactive node-based flow diagrams.

Quart Backend

Fast, async Python web framework for the API and real-time updates.
